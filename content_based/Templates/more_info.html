Phishing Sentinel

Dataset Creation

We used tranco-list.eu's "top-1M.csv" as data source for legitimate website selection and "PhiUSIIL_Phishing_URL_Dataset" as data source for phishing website selection.
Data was collected using the requests library to fetch HTML content of websites.
BeautifulSoup was used to parse and extract relevant features from the HTML.
We have selected a total of 43 features .
The dataset curated by us has totally 37,798 websites ==> 21,266 Legitimate websites and 16,532 Phishing websites.
Data set was created in July 2024.

Model Creation

We experimented with several ML models including Gaussian Naive Bayes, Support Vector Machine, Decision Tree, Random Forest, AdaBoost, Neural Network, and K-Neighbors.
Finally we selected three models, based on parameters of Accuracy,Precision and Recall, We used K-fold Cross validation to calculate these parameters

Random Forest

Random Forest is an ensemble learning method that constructs multiple decision trees during training and outputs the mode of the classes (classification) or the mean prediction (regression) of the individual trees. It improves accuracy and prevents overfitting by averaging multiple independent models.


Random Forest accuracy ==> 0.9625328751860265
Random Forest precision ==> 0.89949467294567
Random Forest recall ==> 0.7854832730692219

Decision Tree

Decision Tree is a supervised learning algorithm used for classification and regression tasks. It partitions the data into subsets based on the value of attributes and recursively splits them until a stopping criterion is met. Each split is made to maximize information gain or minimize impurity, making it interpretable and easy to visualize.


Decision Tree accuracy ==> 0.9481699792624819
Decision Tree precision ==> 0.8171142732449761
Decision Tree recall ==> 0.826265729102901

XGBoost Classifier

XGBoost (Extreme Gradient Boosting) is an optimized gradient boosting algorithm designed for speed and performance. It sequentially builds a series of weak learners (typically decision trees) and combines them to produce a strong learner. XGBoost uses regularization techniques and parallel computing to achieve high accuracy and efficiency in both classification and regression tasks.


XGB Classifier accuracy ==> 0.9194822826318817
XGB Classifier precision ==> 0.8157559780220457
XGB Classifier recall ==> 0.5099583611339051

Django Development

The app is developed using the Django framework. It handles user input, sends requests to the specified URLs, parses the responses, extracts features, and makes predictions using the trained ML models.
We have saved our ML as .pkl files using Joblib module of python and have used the same for prediction process
